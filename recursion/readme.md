# Recursion – Fibonacci

This project implements the Fibonacci sequence using both iterative and recursive approaches.

It was built as part of **The Odin Project – Computer Science section** to reinforce understanding of recursion and problem decomposition.

---

## 📚 What is Fibonacci?

The Fibonacci sequence is a series of numbers where each number is the sum of the two preceding ones:

0, 1, 1, 2, 3, 5, 8, ...

---

## 🧠 Concepts Practiced

- Recursion vs iteration
- Base cases and stopping conditions
- Call stack behavior
- Building results progressively

---

## ⚙️ Features

- Generate Fibonacci sequence of length `n`
- Recursive implementation using function calls
- Iterative implementation for comparison

---

## 🧪 Example

````js
fib(5);
// → [0, 1, 1, 2, 3]

# Merge Sort

This project implements the Merge Sort algorithm using recursion.

Built as part of **The Odin Project – Computer Science section**, focusing on divide-and-conquer algorithms.

---

## 📚 What is Merge Sort?

Merge Sort is a sorting algorithm that:

1. Divides the array into halves
2. Recursively sorts each half
3. Merges the sorted halves

---

## 🧠 Concepts Practiced

- Divide and conquer strategy
- Recursion
- Array manipulation
- Algorithm efficiency

---

## ⚙️ Features

- Sorts arrays of numbers
- Recursive implementation
- Stable sorting algorithm

---

## 🧪 Example

```js
mergeSort([3, 1, 4, 2]);
// → [1, 2, 3, 4]
````
