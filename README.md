# Computer Science Foundations – The Odin Project

This repository contains implementations of core computer science concepts built as part of **The Odin Project** curriculum.

It focuses on strengthening problem-solving skills and understanding fundamental data structures and algorithms through hands-on implementation.

---

## 📚 About The Project

This collection of projects was built to reinforce key computer science principles including recursion, sorting algorithms, data structures, and algorithmic thinking.

The main goal was to:

- Understand how core data structures work internally
- Practice problem decomposition using recursion
- Implement efficient algorithms from scratch
- Improve code organization and readability
- Strengthen debugging and testing skills
- Build a solid foundation for more advanced topics

---

## 🛠️ Built With

- JavaScript (ES6+)
- Jest (Testing)
- Node.js
- Git & GitHub

---

## 📦 Projects Included

### 🔁 Recursion – Fibonacci
- Generates Fibonacci sequences using recursion
- Reinforces base cases and call stack understanding

### 🔀 Merge Sort
- Recursive divide-and-conquer sorting algorithm
- Demonstrates efficient sorting (O(n log n))

### 🔗 Linked List
- Custom implementation of a singly linked list
- Includes insertion, deletion, traversal, and search

### 🗂️ HashMap
- Hash table with collision handling (separate chaining)
- Dynamic resizing based on load factor
- Key-value storage system

### 🌳 Binary Search Tree (BST)
- Balanced tree construction from arrays
- Supports insertion, deletion, traversal, and rebalancing
- Includes height, depth, and balance checking
