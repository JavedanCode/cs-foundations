# Linked List

This project implements a singly linked list data structure.

Built as part of **The Odin Project – Computer Science section** to understand pointer-based structures.

---

## 📚 What is a Linked List?

A linked list is a collection of nodes where each node contains:

- a value
- a reference to the next node

---

## 🧠 Concepts Practiced

- Dynamic data structures
- Node-based relationships
- Traversal
- Mutation (insertion/removal)

---

## ⚙️ Features

- Append and prepend nodes
- Access nodes by index
- Insert and remove at specific positions
- Find values
- Convert list to string representation

---

## 🧪 Example

```js
list.append(1);
list.append(2);
list.append(3);

list.toString();
// → ( 1 ) -> ( 2 ) -> ( 3 ) -> null
```
