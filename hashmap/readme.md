# HashMap

This project implements a hash map (hash table) with collision handling and dynamic resizing.

Built as part of **The Odin Project – Computer Science section**.

---

## 📚 What is a HashMap?

A hash map stores key-value pairs and uses a hash function to determine where data is stored.

---

## 🧠 Concepts Practiced

- Hash functions
- Collision handling (separate chaining)
- Dynamic resizing
- Load factor

---

## ⚙️ Features

- Set and retrieve key-value pairs
- Handle collisions using buckets
- Automatically resize when load factor is exceeded
- Retrieve keys, values, and entries
- Remove and clear data

---

## 🧪 Example

```js
map.set("apple", "red");
map.get("apple");
// → "red"
```
